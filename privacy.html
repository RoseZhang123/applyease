Privacy Policy / 隐私政策

Last Updated: June 2026

English

Overview

ApplyEase is a Chrome extension designed to help users fill university application forms with application information saved in their own browser.

Information We Process

The extension may process application information voluntarily provided by users, including name, email address, phone number, address, educational background, academic records, work or activity history, awards, publications, and other application-related information.

When a user verifies service access, contacts support, or submits feedback, ApplyEase may process the service-related email address, contact information, support message, and a privacy-safe diagnostic summary needed to provide support.

How We Use Information

Application information is used only to save local application records, match application form fields, fill supported university application forms after the user starts the workflow, and improve the reliability of the extension's form automation.

Service access, support, and feedback information is used only for service eligibility checks, customer support, troubleshooting, and necessary follow-up.

Data Storage And Transfer

Application records are stored by default in the user's local Chrome browser storage. ApplyEase does not upload application records to our servers by default.

ApplyEase does not send application records to AI services by default. If future features introduce server sync or AI processing, we will update this policy and the relevant user-facing disclosures before enabling those features.

Service access, support, and feedback information may be transmitted to ApplyEase service endpoints over secure HTTPS connections when needed to provide the requested service.

Third-Party Services

ApplyEase does not sell user data and does not share user data with advertisers or unrelated third parties.

We may use infrastructure providers only as needed to operate the website, service access checks, and support workflow. These providers are not allowed to use user data for advertising or unrelated purposes.

Browser Permissions

  • storage: saves application records and settings in the browser.
  • sidePanel: shows the ApplyEase interface next to supported application pages.
  • scripting: injects form extraction and filling scripts after the user starts the workflow.
  • tabs: identifies the current supported application page and sends commands to the correct tab.
  • downloads: exports user-requested application files or reports.

Host access is limited to supported university application systems and ApplyEase service endpoints declared in the extension manifest. The extension does not request access to all websites.

Contact

If you have any questions about this Privacy Policy, please contact: support@applyease.cn

中文

概述

ApplyEase 是一款 Chrome 扩展，用于帮助用户把保存在自己浏览器中的申请信息填写到学校申请表中。

我们处理的信息

扩展可能处理用户主动提供的申请信息，包括姓名、邮箱、电话号码、地址、教育背景、成绩信息、工作或活动经历、奖项、发表经历以及其他留学申请相关资料。

当用户校验服务访问、联系支持或提交反馈时，ApplyEase 可能处理服务相关邮箱、联系方式、问题描述以及用于排查问题的隐私安全诊断摘要。

信息用途

申请信息仅用于保存本地申请档案、匹配申请表字段、在用户主动开始流程后填写支持的学校申请表，以及提升扩展自动填写功能的可靠性。

服务访问、支持和反馈相关信息仅用于使用资格校验、客服支持、问题排查和必要的后续沟通。

数据存储与传输

申请档案默认保存在用户自己的 Chrome 浏览器本地存储中。ApplyEase 默认不会把申请档案上传到我们的服务器。

ApplyEase 默认不会把申请档案发送给 AI 服务。如果未来功能引入服务器同步或 AI 处理，我们会在启用相关功能前更新本政策和对应的用户可见说明。

在提供用户请求的服务时，服务访问、支持和反馈相关信息可能会通过安全的 HTTPS 连接传输至 ApplyEase 服务端点。

第三方服务

ApplyEase 不会出售用户数据，也不会把用户数据分享给广告商或无关第三方。

我们可能仅在运营网站、服务访问校验和支持流程所必需的范围内使用基础设施服务提供商。这些服务提供商不得将用户数据用于广告或无关目的。

浏览器权限说明

  • storage：在浏览器中保存申请档案和设置。
  • sidePanel：在支持的申请页面旁显示 ApplyEase 操作界面。
  • scripting：在用户开始填写流程后注入表单识别和填写脚本。
  • tabs：识别当前支持的申请页面，并把指令发送到正确标签页。
  • downloads：在用户主动请求时导出申请相关文件或报告。

网站访问权限仅限扩展 manifest 中声明的受支持学校申请系统和 ApplyEase 服务端点。扩展不会请求访问所有网站。

联系方式

如对本隐私政策有任何疑问，请联系：support@applyease.cn
